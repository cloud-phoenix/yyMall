云鱼商城
=========
项目说明：
---------
 云鱼商城：微信小程序商城
 数据来源:淘宝(数据不是很完整，请见谅)


### 目录结构
* components——存放组件
* images——存放图片
* pages——存放页面
* utils——存放工具类
